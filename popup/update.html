Thank you for updating {addon}. You are now running version {version} on Thunderbird {appver}.

New Features and Bug Fixes

  • Attachments and image thumbnails:

    • Esc-r opens the attachment list. Esc-1, esc-2 etc. display attached images 1, 2, ... inside TB (no external app needed).

  • Width of folderbox:

    • The width of the folderbox can be adjusted in about:preferences (Number of characters if foldernames are long).

  • Bug fixes:

    • Select final folder by enter - Fix autocomplete in rules window - Put instructions into tooltip to gain space.

  • Create new rule, Convert last email move into rule, Apply rule:

    • A new rule can now be created from the main window. The folderset used for the last email move can be inserted into a new rule. In the rule list, the rules can be directly applied without opening the folders in Thunderbird's main view.

  • Composer: Switch to cc or bcc line

    • By keyboard, esc followed by c, the cc line is shown and focused. Similiarly, esc - b opens the bcc , esc-t the to line. Just as in earlier versions, cc (+space), bcc (+space), to (+space) can be used as well (disabled in 2.2.3).

  • Composer: Always show cc or bcc line

    • In the preferences, you can choose to have cc and/or bcc line always visible
